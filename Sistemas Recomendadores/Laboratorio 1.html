Instalación¶

Para facilitar el proceso de instalacin de esta actividad, trabajaremos con una mquina virtual que tendr casi todos los programas instalados.

Como motor de mquinas virtuales usaremos Virtual Box. Desde ese link tendrn que descargar la versin que mejor se ajuste a su sistema operativo. Luego desde los servidores de la universidad deben descargar la mquina virtual ya configurada.

Finalmente debemos importar la mquina descargada dentro de Virtual Box, para ello deben seguir los siguientes pasos: Abrir virtual box > Archivo > Abrir servicio virtualizado, o bien Crtl+I.

Observacin: la contrasea del usuario configurado es ubuntu.

Una vez abierto la mquina virtual, abrir la consola o terminal y escribir los siguiente comandos:

$ sudo apt-get install git
$ sudo pip3 install jupyter
$ sudo pip3 install matplotlib

Descargar este proyecto ya sea con git o mediante el botn de descargar y ejecutar notebook.

$ git clone https://github.com/stgolarrain/recsys-labs.git
$ cd recsys-labs/assignment-1
$ jupyter notebook

Instrucciones¶

En este laboratorio exploraran la herramienta de sistemas de recomendacion pyreclab.

La biblioteca pyreclab es una librera que implementa una serie de tcnicas y modelos de recomandacin, exponiendo una interfaz de fcil uso manteniendo un buen rendimiento en meoria y en CPU.

En este laboratorio ejecutaremos los scripts de la carpeta examples de la librera y evaluaremos cmo se comportan las mtricas RMSE y MAE, frente a diferentes parmetros.

La librera puede ser instalada ejecutando el siguiente comando en la consola:

$  pip install pyreclab

Para facilitar la experiencia del laboratorio los archivos de ejemplo han sido guardados en la subcarpeta src.
